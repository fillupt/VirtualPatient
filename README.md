# VirtualPatient

This is the build repository for the Virtual Patient. Have a play with it, and email bugs and suggestions to Phil at p.turnbull@auckland.ac.nz

Version 1.3

Bug fixes:
Text invisible on Cover Test test mode.

New features:

Hair!
